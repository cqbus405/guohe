### 产品中心

> GET localhost:3001/api/guohe/products

**参数**

* page：页码，从1开始。每页返回10条数据
* type：商品类型

type | 说明
-----|------
1    |  全部
2    |  气泡
3    |  气垫
4    |  塑料
5    |  其他
